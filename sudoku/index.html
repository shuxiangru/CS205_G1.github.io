Simple 9 x 9 Sudoku

Sudoku is a combinatorial number placement puzzle. The objective of this game is to fill a 9×9 grid (board) with digits so that each column, each row, and each of the nine 3×3 subgrids (boxes) contains all of the digits from 1 to 9. Higher-rank sudokus usually have n^2*n^2 board with n*n boxes (n>3). However, higher-rank sudokus take hours to run in serial. Thus, we aim to parallelize a sudoku solver algorithm with several parallelization techniques (OpenMP and MPI), so that giant sudoku solver can be solved within a limited time.

Giant Sudoku Problem

  • n2×n2 board with n×n boxes (n>3) (e.g. 16*16, 25*25, 100*100)
  • Why do we need high throughput computing tools?
    • Execution time increases drastically with n
    • High-rank problems with a large number of empty cells can take hours or even days to solve

Existing Solutions

  • Theoretical speedup of Sudoku solver with OpenMP is linear
  • Academic paper on MPI exists, but no implementation found
